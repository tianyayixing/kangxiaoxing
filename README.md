# 康小星

## 📱 项目简介

这是一款专注于个人健康饮食管理的智能Web应用，帮助用户制定科学的一周饮食计划，实现健康饮食管理目标。

## ✨ 核心功能

### 1. 首次引导设置
- 3步式简易配置
- 饮食习惯选择（素食/肉食/杂食/喜辣/清淡）
- 饮食目标设定（减脂/增肌/维持健康/控糖）
- 健康状况填写
- 忌口食材管理

### 2. 每日饮食计划
- 查看今日饮食安排
- 实时营养评分（0-100分）
- 热量、蛋白质、碳水等营养素统计
- 营养素比例可视化
- 智能建议提示

### 3. 周饮食计划管理
- 周一至周日完整规划
- 早餐/午餐/晚餐/加餐分类
- 100+种菜品数据库
- 菜品分类浏览（早餐/主食/蔬菜/肉类等）
- 菜品搜索功能
- 快速添加/删除菜品
- 实时营养分析

### 4. 菜品详情
- 详细营养成分（热量、蛋白质、脂肪、碳水、纤维、维生素）
- 菜品功效说明
- 烹饪方法指导
- emoji图标展示

### 5. 智能营养分析
- 实时计算每日营养摄入
- 营养均衡性评估
- 热量控制建议
- 蛋白质/脂肪/碳水比例分析
- 膳食纤维和维生素评估
- 个性化优化建议

### 6. 周营养报告
- 每周数据统计
- 可视化图表展示：
  - 每日热量摄入趋势图
  - 营养素比例饼图
  - 每日健康评分柱状图
- 平均评分/热量/蛋白质统计
- 每日详细数据回顾

### 7. 数据安全
- 本地加密存储
- 无需登录
- 数据完全保存在本地
- 隐私安全保障

## 🛠️ 技术栈

- **前端框架**: React 18
- **构建工具**: Vite
- **UI组件**: Ant Design Mobile
- **图表库**: ECharts
- **数据加密**: CryptoJS
- **本地存储**: localStorage (加密)

## 📦 安装与运行

### 环境要求
- Node.js >= 16
- npm >= 7

### 安装步骤

1. 克隆项目
```bash
cd diet-manager
```

2. 安装依赖
```bash
npm install
```

3. 启动开发服务器
```bash
npm run dev
```

4. 构建生产版本
```bash
npm run build
```

5. 预览生产版本
```bash
npm run preview
```

## 🎯 使用指南

### 首次使用
1. 打开应用后会进入引导页
2. 完成3步设置：
   - 选择饮食习惯
   - 设定饮食目标
   - 填写健康状况和忌口

### 制定饮食计划
1. 点击"周计划"标签
2. 选择星期几
3. 点击"添加"按钮
4. 从菜品库中选择菜品
5. 查看实时营养分析和建议

### 查看营养报告
1. 点击"报告"标签
2. 查看本周营养统计
3. 查看可视化图表
4. 了解每日详细数据

## 📊 菜品数据库

应用内置100+种常见菜品，涵盖：
- 早餐类：煎蛋、牛奶、全麦面包、燕麦粥等
- 主食类：米饭、馒头、面条、红薯、玉米等
- 蔬菜类：西蓝花、菠菜、胡萝卜、西红柿等
- 肉类：鸡胸肉、牛肉、鱼肉、虾等
- 豆制品：豆腐、豆腐干等
- 水果类：苹果、香蕉、橙子、葡萄等
- 家常菜：番茄炒蛋、青椒肉丝、麻婆豆腐等
- 汤类/粥类/凉菜等

每个菜品包含：
- 热量（卡路里）
- 蛋白质、脂肪、碳水化合物含量
- 膳食纤维、维生素C、B族维生素
- 功效说明
- 烹饪方法
